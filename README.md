# Library Management System

A full-stack Library Management System built with **React (frontend)** and **Django (backend)**.

---

## Features

- User authentication (Admin, Librarian, User)  
- Browse and search books  
- Borrow and return books  
- Admin and Librarian dashboards with statistics  
- Real-time book availability updates
